## 🎨 CORE PRINCIPLES (APPLIED EVERYWHERE)

### 1. ONE Hero Per Screen

```
✅ Home: Wallet balance (dark gradient)
✅ Services: Featured service (regional)
✅ Bookings: Today's trip (if exists)
✅ Rewards: Points balance (dark)
✅ Profile: User identity (dark)
✅ Confirmation: Checkmark (dark circle)
✅ Login: Checkmark (dark circle)

❌ No other gradients anywhere
```

**Rule:** If you see 2+ gradients → FIX IT.

---

### 2. Outcome Language (Not Feature Language)

```
❌ BEFORE (Feature-focused):
- "AI Travel Assistant"
- "Quick Pay Feature"
- "Booking Management System"
- "Redemption Portal"
- "Security Settings"

✅ AFTER (Outcome-focused):
- "Msaada wa Safari" (Travel Help)
- "Tuma Pesa" (Send Money)
- "Safari Zangu" (My Trips)
- "Tumia Pointi" (Use Points)
- "Usalama" (Security)
```

**Rule:** If it describes HOW → Rewrite to WHAT YOU GET.

---

### 3. Trust Felt (Not Stated)

```
❌ BEFORE (Trust announced):
- "Secure • Encrypted • Trusted" badges
- Lock icons everywhere
- "Bank of Tanzania Approved" banner
- Security checkboxes

✅ AFTER (Trust felt):
- Predictable behavior
- Always show receipt
- Official-looking formats
- Calm error handling
- Consistent motion
```

**Rule:** Remove trust badges. Build trust through predictability.

---

### 4. Calm > Flashy

```
❌ BEFORE (Flashy):
- Multiple colors competing
- Bounce animations
- Confetti on success
- Celebratory language
- Gamification badges

✅ AFTER (Calm):
- Dark gradients (premium)
- Ease-out only (no bounce)
- Simple checkmarks
- Factual language
- Progress bars (not badges)
```

**Rule:** Fintech = boring is good. Celebration = distrust.

---

### 5. Intelligence Hidden

```
❌ BEFORE (Explicit AI):
- "AI Travel Assistant"
- "Smart recommendations powered by AI"
- AI icons
- Tech jargon

✅ AFTER (Invisible):
- "Msaada wa Safari"
- "Mapendekezo kulingana na safari zako"
- Zero AI labels
- Just works
```

**Rule:** If you mention "AI" → REMOVE IT. Users feel it's smart without labels.

### Step 2: Update Payment Flow (3 min)

```tsx
// After successful payment
const [showConfirmation, setShowConfirmation] = useState(false);
const [paymentDetails, setPaymentDetails] = useState(null);

const handlePaymentSuccess = (details) => {
  setPaymentDetails(details);
  setShowConfirmation(true);
};

// Render confirmation
{showConfirmation && (
  <PaymentConfirmationWorldClass
    amount={paymentDetails.amount}
    recipient={paymentDetails.recipient}
    type="send" // or 'bill', 'purchase'
    reference={paymentDetails.reference}
    timestamp={new Date()}
    onDone={() => {
      setShowConfirmation(false);
      setCurrentPage('dashboard');
    }}
    onViewReceipt={() => {
      setCurrentPage('receipt');
    }}
    language="sw"
  />
)}
```
