## ⚡ FASTEST METHOD (2 Minutes)

### **If you DON'T have GitHub:**

#### **Step 1: Go to Replit**
👉 **Click here:** https://replit.com

#### **Step 2: Upload Files**
1. Click **"+ Create Repl"**
2. Choose **"Node.js"** template
3. Name it: `gopay-tanzania`
4. **Drag & drop ALL your project files** into Replit
   - Make sure to include:
     - ✅ `.replit` file
     - ✅ `package.json`
     - ✅ `vite.config.ts`
     - ✅ `index.html`
     - ✅ All `/components` folder
     - ✅ All files!

#### **Step 3: Click "Run"**
- Big green button at top
- Wait 30 seconds
- Your app is LIVE! 🎉

**Your URL:** `https://gopay-tanzania.your-username.repl.co`

## 🆘 TROUBLESHOOTING

### **Problem: Files not uploading**
- Zip all files
- Upload zip file
- Replit auto-extracts

### **Problem: "npm install failed"**
Run in Shell:
```bash
npm install --legacy-peer-deps
```

### **Problem: Blank screen**
- Press F12 (browser console)
- Check for errors
- Make sure `index.html` exists

### **Problem: Port error**
```bash
pkill -f vite
npm run dev
```
