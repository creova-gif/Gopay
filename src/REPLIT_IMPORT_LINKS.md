   - ✅ Detects `.replit` config
   - ✅ Runs `npm install`
   - ✅ Sets up environment

## 🆘 Troubleshooting Import Issues

### **Issue: "Repository not found"**
- Make sure repo is public
- Or authorize Replit to access private repos
- Check URL spelling

---

### **Issue: "No .replit file"**
Replit creates one automatically, but you can specify:

Create `.replit` manually:
```
run = "npm run dev"
entrypoint = "index.html"
```

---

### **Issue: "Dependencies failed to install"**
In Replit Shell:
```bash
npm install --legacy-peer-deps
```

---

### **Issue: "Port not accessible"**
Check `vite.config.ts`:
```typescript
server: {
  host: '0.0.0.0',
  port: 5173
}
```
